Overview

Welcome to this demo movie recommender application. This demo has been installed with approximately four thousand movies and 500,000 ratings. The ratings have been generated randomly. The purpose of this web application is to allow users to search for movies, rate movies, and receive recommendations for movies based on their ratings.

To get started:

    {% if not current_user.is_authenticated %}
  • New users: Register for an account
  • Existing users: Login to your account
    • {% endif %}
  • Use the Search box in the header to search for and rate some movies
  • Get Recommendations for movies

NOTE: The goal of this webapp is to show Apache Spark Machine Learning. Therefore, some functionality is missing and it doesn't have the polish that you would expect from a commercial movie recommendation web site.

Running your own demo instance

 You can deploy your own instance of this web application (plus Cloudant and Compose Redis). Click on the button and follow the instructions. CAUTION: a python flask application instance with 128MB memory and an instance of Cloudant 'Lite' and an instance of Compose for Redis 'Standard' will get deployed in your Bluemix account - you may get charged for these services. Please check charges before deploying.

After deploying to Bluemix, you will need to create a new DSX project and import the notebooks (see the Source Code section below). The notebook Step 07 - Cloudant Datastore Recommender.ipynb is responsible for creating recommendations and saving the recommendations to Cloudant. You will not get recommendations until you have run this notebook.
